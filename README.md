# CRUD Simples Java

Projeto desenvolvido para praticar:

- POO
- Classes e Objetos
- Encapsulamento
- Organização em pacotes
- Git e GitHub

## Funcionalidades

- Cadastro de usuários (em memória)
- Leitura de usuários (em memória)
- Atualização de usuários (em memória)
- Exclusão de usuários (em memória)
